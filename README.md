# Open Source Audit Project

Name: Deepak Kumar

## Introduction
This project is created as part of the Open Source Software course. It performs system audit tasks using shell scripting.

## Scripts Description

- script1.sh : Displays system information
- script2.sh : Checks if a package is installed
- script3.sh : Shows disk usage and largest files
- script4.sh : Analyzes log files
- script5.sh : Generates final audit report

## How to Run

chmod +x *.sh

./script1.sh
./script2.sh git
./script3.sh
./script4.sh sample.log
./script5.sh

## Output
The final report is saved in audit_report.txt

## Conclusion
This project shows how shell scripting can be used for basic system auditing.
